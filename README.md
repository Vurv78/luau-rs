> High level [Luau](https://github.com/Roblox/luau) bindings for the [Rust](https://www.rust-lang.org) programming language. Mini [rlua](https://github.com/amethyst/rlua) but for luau.

## Usage
Add this to your ``Cargo.toml``
```toml
[dependencies]
rluau = { version = "0.1.0" }
```
